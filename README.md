# 基于 Express 的轻量级 RESTful API 服务器框架


## 目标

+ 尽可能少编写业务无关的代码
+ 尽可能共享公共部分代码
+ 强类型检查
+ 单元测试
+ 自动根据代码生成精准的API文档
+ 保证最大的灵活性，除基本框架之外，不限制用户使用何种数据库客户端或ORM
+ 支持ES7的async function
